# 手順

1\. SLコマンドのインストール
```
apt install sl
```
2\. 蒸気機関車を発射
```
sl
```
2\. 空飛ぶ蒸気機関車を発射
```
sl -F
```
3\. 小さい蒸気機関車を発射
```
sl -l
```
4\. 空飛ぶ小さい蒸気機関車（アクシデントあり）を発射
```
sl -alF
```
